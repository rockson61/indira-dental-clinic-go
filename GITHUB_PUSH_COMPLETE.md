## 🆘 **TROUBLESHOOTING**

### **Q: Push failed - Authentication error**
**A:** Make sure your SSH key is added to GitHub:
```bash
# Check if SSH key exists
ls ~/.ssh/id_rsa.pub

# If not, generate one:
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

# Add to GitHub:
cat ~/.ssh/id_rsa.pub
# Copy and add to: https://github.com/settings/keys
```
